
# IOT - tecnologias disruptivas

👌**Acesse aqui:**
[IoT](https://baccarinha.github.io/iot/)

🎯 Objetivo

Criar uma aplicação web educativa capaz de demonstrar o funcionamento da Internet das Coisas, apresentando seus principais conceitos, componentes e aplicações práticas no cotidiano e na indústria.

|🧩 Funcionalidades|
|----|
📡 Apresentação do conceito de Internet das Coisas (IoT)
🧠 Explicação dos principais componentes de um sistema IoT
🌍 Demonstração de aplicações reais da IoT
🖥️ Interface informativa e organizada
📱 Layout responsivo para diferentes dispositivos
📖 Conteúdo educativo sobre tecnologia moderna
🛠️ Tecnologias Utilizadas

|Este projeto foi desenvolvido utilizando as seguintes tecnologias:|
|----|
HTML5 — Estruturação das páginas
CSS3 — Estilização e layout visual
JavaScript — Interatividade e dinamismo
GitHub Pages — Hospedagem do projeto
Inteligência Artificial (IA) — Apoio na criação da estrutura e ideias do projeto
